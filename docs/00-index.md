## Session Highlights

### 🚀 [01 | Setup](01-setup.md)
- **Environment**: Installing the Copilot extension and authenticating.

### ✨ [02 | Basics](02-basics.md)
- **Ghost Text**: Interacting with proactive autocompletion.
- **Next Edit Suggestions**: Predicting the next logical change in your file.
- **Interfaces**: Difference between Inline Chat (`Ctrl + I`) and Chat Panel (`Ctrl + Alt + I`).

### 🛠️ [03 | Customization](03-customization.md)
- **Personalization**: Configuring commit message rules and style guides.
- **Gitflow**: Implementing semantic commit messages (feat, fix, chore) using AI.

### 📋 [04 | Project Instructions](04-project-instructions.md)
- **Global Context**: Using `.github/copilot-instructions.md` to enforce PEP 8, type hints, and Pydantic validation.
- **Consistency**: Ensuring Copilot understands your business logic and architecture from the start.

### 🤖 [05 | Advanced](05-advanced.md)
- **Custom Agents**: Building specialized AI personas (e.g., "Pydantic Architect").
- **Skills**: Defining technical capabilities that agents can invoke for specialized tasks.

### 🔌 [06 | MCP](06-mcp.md)
- **Extensibility**: Integrating Copilot with external data sources and APIs via Model Context Protocol.
- **Tooling**: Configuring servers to let AI browse databases or call enterprise services.

### 📚 [07 | Resources](07-resources.md)
- **Documentation**: Direct links to official guides.
- **Continuous Learning**: Recommended practices for staying current with AI tools.
