# 07 - Resources and Documentation

To continue deepening your knowledge of GitHub Copilot and stay up to date with the latest news, we recommend the following resources.

## 1. Official Documentation
- [GitHub Copilot General Documentation](https://docs.github.com/en/copilot)
- [GitHub Copilot in VS Code](https://code.visualstudio.com/docs/copilot/overview)
- [GitHub Copilot CLI](https://docs.github.com/en/copilot/github-copilot-in-the-cli/using-github-copilot-in-the-cli)

## 2. Practical Learning
- [GitHub Skills](https://skills.github.com/): Free interactive courses within GitHub repositories.
- [Copilot for Individuals, Business, and Enterprise](https://github.com/features/copilot/plans): Plan comparison.

## 3. Community and News
- [GitHub Blog - AI Category](https://github.blog/category/ai/): Announcements of new features (Agents, MCP, etc.).
- [VS Code Changelog](https://code.visualstudio.com/updates): Every month VS Code adds improvements in Copilot integration.
